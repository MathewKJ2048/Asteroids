# Asteroids
A two-player version of asteroids, written using pygame.

Competitive and co√∂perative mode are both supported.

## Controls:

### Player 1:
`i`   accelerate  
`k`   decelerate  
`l`   rotate clockwise  
`j`   rotate counterclockwise  
`m`   shoot  
`,`   fire missile  

### Player 2:
`w`   accelerate  
`s`   decelerate  
`d`   rotate clockwise  
`a`   rotate counterclockwise  
`z`   shoot  
`x`   fire missile
